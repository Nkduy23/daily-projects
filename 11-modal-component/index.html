API Reference
Cách sử dụng Modal API
JavaScript API
// Mở modal theo ID
Modal.open('my-modal');
Modal.open('my-modal', { backdropClose: false });

// Đóng modal
Modal.close('my-modal');
Modal.closeAll();

// Alert shortcuts
Modal.alert('success', 'Tiêu đề', 'Nội dung');
Modal.alert('warning' | 'error' | 'info', ...);

// Confirm dialog
Modal.confirm('Tiêu đề', 'Nội dung', {
  confirmText: 'OK',
  cancelText: 'Hủy',
  danger: true,
  onConfirm: () => { /* ... */ },
  onCancel:  () => { /* ... */ },
});

// Auto-close
Modal.openAutoClose('modal-id', 3000); // 3 giây

// Events
document.addEventListener('modal:open',  e => console.log(e.detail.id));
document.addEventListener('modal:close', e => console.log(e.detail.id));
HTML Structure
<div class="modal-backdrop" data-modal-id="my-modal">
  <div class="modal modal-sm | modal-lg | modal-full"
       data-anim="scale | slide | zoom | flip">
    <div class="modal-header">
      <div class="modal-title">Tiêu đề</div>
      <button class="modal-close">✕</button>
    </div>
    <div class="modal-body">Nội dung...</div>
    <div class="modal-footer">
      <button class="btn-ghost modal-close">Hủy</button>
      <button class="btn-primary">Xác nhận</button>
    </div>
  </div>
</div>
Modal mặc định

Đây là modal cơ bản với đầy đủ header, body và footer. Modal hỗ trợ đóng bằng nút ✕, phím Escape hoặc click ra ngoài backdrop.

Thử nhấn Escape để đóng modal này.

✨ Fade + Scale

Modal xuất hiện với hiệu ứng fade kết hợp scale nhẹ. Đây là animation mặc định, phù hợp với hầu hết use case.

🚀 Slide Up

Modal trượt từ dưới lên trên. Animation này rất phù hợp cho mobile — giống như bottom sheet trong iOS/Android.

🔍 Zoom

Modal phóng to từ kích thước nhỏ. Phù hợp cho dialog xác nhận, thông báo quan trọng cần thu hút sự chú ý.

🎴 Flip 3D

Hiệu ứng lật 3D ấn tượng. Dùng cho các thao tác đặc biệt như reveal, card flip, hoặc khi cần tạo điểm nhấn thị giác mạnh.

Điều khoản dịch vụ

1. Giới thiệu

Bằng cách sử dụng dịch vụ của chúng tôi, bạn đồng ý với các điều khoản này. Vui lòng đọc kỹ trước khi tiếp tục.

2. Quyền riêng tư

Chúng tôi cam kết bảo vệ quyền riêng tư của bạn. Dữ liệu cá nhân sẽ không được chia sẻ với bên thứ ba mà không có sự đồng ý của bạn.

3. Sử dụng hợp lệ

Bạn đồng ý không sử dụng dịch vụ cho các mục đích bất hợp pháp hoặc vi phạm quyền của người khác.

4. Giới hạn trách nhiệm

Dịch vụ được cung cấp "như hiện tại". Chúng tôi không đảm bảo tính liên tục hay không có lỗi của dịch vụ.

5. Thay đổi điều khoản

Chúng tôi có quyền thay đổi điều khoản bất cứ lúc nào. Thông báo sẽ được gửi qua email trước 30 ngày.

6. Liên hệ
